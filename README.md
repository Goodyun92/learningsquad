# learningsquad
PDF 문서 기반 LLM 문제 생성 및 채점 웹 서비스
> 사용자가 업로드한 문서로부터 서술형 문제와 모범 답안을 생성하고, 사용자의 답변을 입력받아 모범
> 답안을 기준으로 채점을 제공하는 LLM 기반 웹서비스 및 이를 활용하여 사용자가 학습할 수 있는
> 서비스를 제공하는 웹서비스

</br>

#### Development Environment
- Jdk 17
- Spring Boot 3.3.7
- MySQL
